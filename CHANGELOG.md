# Changelog

## v1.0.7

发布日期: 2026-03-25

新增内容:

- 新增语音识别连接配置板块，统一管理 App ID、Access Token、识别语言、热词、标点和延迟停顿等选项。
- 新增 LLM 翻译连接配置，支持单独设置 Base URL、API Key、翻译模型、提示词和目标语言。
- 新增 OpenAI 兼容模型列表拉取能力，可直接获取可用翻译模型并在界面中选择。
- 新增翻译输出开关与翻译快捷键，识别结果可在翻译完成后再输出，避免旧文本提前上屏。
- 新增识别与翻译状态提示优化，包括浮动提示、翻译中状态和连接状态反馈。

说明:

- 本版本基于当前 `master` 分支发布，对应提交: `e5e0cf7837c717edb8ad14d0c6013cc11f4c4f9e`
- 相对 `v0.1.6` 的主要变化为翻译与识别配置能力增强，以及相关输出流程优化。
